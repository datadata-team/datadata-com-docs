1. **上传数据源**  
用户可将自有数据上传至平台，并与系统自带的数据源进行联动分析。支持灵活的数据融合，满足个性化研究需求。

![image](./data2.png)

2. **连接外部数据源**  
DATADATA支持连接多种外部数据库，包括 **MySQL**、**Postgres**、**Clickhouse**、**Trino** 等。通过将外部数据源与平台结合，用户能够利用平台强大的数据分析能力，对自有数据库进行深度挖掘与分析。

![image](./data3.png)
