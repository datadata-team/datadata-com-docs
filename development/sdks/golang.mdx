通过我们的 Go SDK 可以帮助您快速将 datadata 与您的 Go 应用程序集成。

使用下面的命令安装 SDK 依赖。

```shell
go install github.com/datadata-team/datadata-sdk-go@master
```

## Modules

### API Key

api-key 模块提供接口认证相关的功能。
