Javascript SDK 适用于 Browser 和 NodeJS 环境。

使用下面的命令添加 Javascript SDK 到您的项目依赖。

<Tabs>
<TabItem value="npm" label="npm" default>
```shell
npm install @datadata/sdk-javascript
```
</TabItem>
<TabItem value="pnpm" label="pnpm">
```shell
pnpm add @datadata/sdk-javascript
```
</TabItem>
<TabItem value="yarn" label="yarn">
```shell
yarn add @datadata/sdk-javascript
```
</TabItem>
</Tabs>
