Datadata 提供的嵌入组件可以让您的 Web 应用快速集成 Datadata 的核心功能。

嵌入组件通过原生 [Web-Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) 构建，
使用嵌入组件就像使用原生 HTML 标签一样简单，并且可以于任何前端框架无缝集成。

:::tip
嵌入组件依赖 API 接口，在使用嵌入组件之前，需要准备好 API-Token，详细信息参见[身份认证](../overview/authentication)章节。
:::

## 嵌入编辑器

嵌入编辑器组件可以让您嵌入一个交互式的查询组件，快速让您的 Web 应用集成数据分析、可视化功能。

嵌入编辑器包含的功能由：

1. 数据目录查看
2. SQL 查询功能
3. 可视化功能
4. 列表功能
5. CSV 文件上传功能

更多详细信息，参考 [嵌入编辑器](./embedded-editor) 介绍章节。

## 嵌入查看器

有时候，您可能想在文章页面插入已经编辑好的可视化图表，此时只需要查看功能，这就是查看组件的用武之地。

更多详细信息，参考 [嵌入查看组件](./embedded-viewer) 介绍章节。
